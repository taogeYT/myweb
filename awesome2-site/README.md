# blog-python-app

> 1.项目说明  

项目来自廖雪峰的官网，目的是学习写一个自己的web微框架

如果想学习python 他的教程值得推荐 [廖雪峰官网](http://www.liaoxuefeng.com/)



> 2.项目结构

```
awesome-python-webapp/   <-- 根目录
|
+- backup/               <-- 备份目录
|
+- conf/                 <-- 配置文件
|
+- dist/                 <-- 打包目录
|
+- www/                  <-- Web目录，存放.py文件
|  |
|  +- static/            <-- 存放静态文件
|  |
|  +- templates/         <-- 存放模板文件
|
+- LICENSE               <-- 代码LICENSE
```
